- Practiced with array prototypes

```
1. array.prototype.filter()
2. array.prototype.map()
3. array.prototype.sort()
4. array.prototype.reduce()
```

**Thoughts:** 
Today’s challenge was another great stretch and contained good examples of the power of array prototypes. One const was an array of inventors with objects containing their first name, last name, birthdate, and death date. The second array contained a list of people’s first and last names. 
 
Using these array prototypes allowed me to get data in many different ways. This was another challenge in which the concepts made sense, but it’ll take a lot of practicing getting proficient with setting these functions up (especially with the ES6 syntax). 
 
Here is an example of finding all inventors born in the 1500s.
 
```
const fifteen = inventors.filter(inventors => (inventors.year >= 1500 && inventors.year < 1600));
    console.table(fifteen);
 
```
- The variable 'const' was named fifteen and was assigned to a function.
- 'inventors.filter' is utilizing Javascript’s array.prototype.filter() to gather this data. 
- 'inventors =>' is using this lambda to create a function instead of using the ES5 way
- '(inventors.year >= 1500 && inventors.year < 1600)' is selecting everyone in the 1500s
- 'console.table(fifteen);' creates a nice table of the data! I hadn’t heard of console.table before.
